# Version Control ບໍ່ຕ້ອງລົງໂປຣແກຣມ (ໃຊ້ GitHub ເວັບ)

> ບໍ່ຕ້ອງ install ຫຍັງລົງເครื่องไหนเลย. GitHub.com ມີ version history ໃນຕัวอยู่แล้ว.

---

## ໂຄງສ້າງລະບົບ (ເข้าใจก่อน)

```
┌─ ຄอมพนักงาน (ຫลายเครื่อง) ─┐     ← แค่เปิดเว็บ (Pages URL) ไม่ต้องลงอะไร
│  Chrome → เปิด POS         │
└────────────┬───────────────┘
             │ ดึง code จาก
             ▼
   GitHub Pages (ເວັບໄຊ)          ← deploy อัตโนมัติเมื่อ commit
             ▲
             │ อัปไฟล์ + Commit
   GitHub.com repo               ← ★ version history อยู่ที่นี่ ★
             ▲
             │ ก๊อปจาก
   ໂฟลเดอร์ Cowork (ผมดูแล)       ← แก้ code ที่เดียว = ตัวจริง
```

**กฎเดียว:** code แก้ที่ **โฟลเดอร์ Cowork** ที่เดียว → อัปขึ้น GitHub → commit. พนักงานแค่เปิดเว็บ.

---

## วิธีอัปเดต (บนเว็บ github.com — ไม่ต้องลงอะไร)

1. เข้า https://github.com/bkiasystem-dotcom/bokeo-airport-pos
2. **Add file → Upload files**
3. ลากไฟล์ล่าสุดทั้งหมดจากโฟลเดอร์ Cowork:
   `index.html` · `app.js` · `database.js` · `styles.css` · `html2pdf.bundle.min.js` · `logo.png`
   (ไฟล์ชื่อซ้ำ = ทับของเก่าให้เอง)
4. ล่างสุด ช่อง **Commit changes** → ใส่ข้อความ (ເຊັ່ນ `v15 - PDF + logo`) → กด **Commit changes**
5. รอ ~1 นาที GitHub Pages อัปเดตเอง → พนักงานกด Ctrl+Shift+R

> ✅ ทุกครั้งที่ Commit = เซฟ 1 เวอร์ชันอัตโนมัติ

---

## ดูประวัติ / ย้อนเวอร์ชัน (เมื่อไฟล์พัง)

**ดูประวัติทั้งหมด:**
repo → กด **Commits** (ไอคอนนาฬิกา ใกล้ปุ่ม Code) → เห็นทุกเวอร์ชัน

**ย้อนกลับทั้ง commit:**
Commits → เปิด commit ที่พัง → ปุ่ม **• • •** → **Revert** → Commit

**กู้ไฟล์เดียวจากเวอร์ชันเก่า:**
1. เปิดไฟล์ (ເຊັ່ນ app.js) → กด **History**
2. เลือกเวอร์ชันที่ใช้งานได้ → กด **• • • → View file** → **Raw** → ก๊อปเนื้อหา
3. กลับมาแก้ไฟล์ → วางทับ → Commit

---

## กฎเหล็ก (กันไฟล์ regress / ขาดท้าย แบบที่เคยเจอ)

1. **อัปจากโฟลเดอร์ Cowork เท่านั้น** — อย่าเอาไฟล์ที่ดาวน์โหลดเก่ามาอัปทับ
2. **อัปครบทุกไฟล์พร้อมกันใน 1 commit** — อย่าอัปทีละไฟล์แบบครึ่งๆ
3. หลังอัป → เปิดเว็บจริง เช็คว่าใช้ได้ → ถ้าพัง กด Revert ทันที
4. ตั้งชื่อ commit ให้รู้ว่าทำอะไร — จะหาเวอร์ชันย้อนง่าย

---

## คอมพนักงานเครื่องใหม่ (ไม่ต้องลงอะไร)
แค่เปิด Chrome → เข้า URL ของ GitHub Pages → ใช้ได้เลย.
ข้อมูลร่วม (รายชื่อ, QR, รูป, rate) ดึงจาก Google Sheet + Drive อัตโนมัติ.
